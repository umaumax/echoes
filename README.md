# echoes

echo server

## how to install
``` bash
cargo install --git https://github.com/umaumax/echoes
```

## how to use
``` bash
echoes 0.1.0

USAGE:
    echoes [FLAGS] [addr]

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information
        --verbose    verbose flag

ARGS:
    <addr>    bind address [default: localhost:8080]
```
